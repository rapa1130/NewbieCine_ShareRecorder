# 뉴비시네
낯선 타인과 대화하는 영화관이라는 브랜드 컨셉을 전달하는 프로그램.


## 프로젝트 소개
서울과학기술대학교 시각디자인학과 아이덴티티 트랙 졸업전시로 제작한 개인 프로젝트입니다. 
낯선 타인과 이야기하는 영화관이라는 브랜드의 컨셉을 전시현장에서 체험형으로 전달하기 위해 Unity로 제작하였습니다.
영화에 대한 질문을 보고 대답을 녹음할 수 있고 다른 이들이 녹음한 대화를 터치하여 들어볼 수 있습니다.


## 개발 기간
2025.10.14 ~ 2025.11.02 (3주)

## 개발멤버
안병헌 개인작업입니다.

## 개발환경
- Unity 2023.2.20f1
- Visual Studio

기기는 IPad로 전시되었고 무선 헤드폰을 통해 사운드 입출력을 진행하였습니다.
