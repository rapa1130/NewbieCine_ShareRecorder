**'낯선 타인과 대화하는 영화관'** 이라는 브랜드 컨셉을 전달하는 프로그램.

서울과학기술대학교 시각디자인학과 아이덴티티 트랙 졸업전시로 제작한 개인 프로젝트입니다. 
낯선 타인과 이야기하는 영화관이라는 브랜드의 컨셉을 전시현장에서 체험형으로 전달하기 위해 Unity로 제작하였습니다.
영화에 대한 질문을 보고 대답을 녹음할 수 있고 다른 이들이 녹음한 대화를 터치하여 들어볼 수 있습니다.

## 주요 기능
###### 녹음
하단에 녹음버튼을 터치하면 녹음이 시작됩니다.
녹음버튼을 다시 터치하면 녹음이 종료되며 해당 음성 녹음을 재생할 수 있는 재생버튼이 생성됩니다.
불필요한 녹음 파일을 방지하기위해 녹음 시간이 3초 이하일 경우 저장되지 않고 안내메세지를 잠깐 띄웁니다.

###### 재생
재생 버튼들을 클릭하여 여태까지 녹음된 음성들을 들어볼 수 있습니다. 
한 번에 한 버튼만 재생할 수 있고 특정 음성이 재생되는 도중 다른 재생 버튼을 클릭하면 기존 재생은 종료되고 클릭한 버튼이 재생됩니다.  재생 버튼들 사이 선을 연결하여 브랜드 컨셉인 별자리를 표현합니다.

###### 스크롤
좌우로 스크롤 하여 여태까지 녹음된 다른 음성들도 들어볼 수 있습니다.

## 주요 기술

###### 스크롤링에 따른 로딩,언로딩
전시기간동안 녹음된 음성이 많이질 가능성을 고려하여 화면 기준으로 좌우 30개의 버튼만 로딩하도록 하였습니다.
음성 녹음은 파일 단위로 관리되고 화면 스크롤에 따라 30개를 넘어서는 방향엔 버튼들을 언로딩, 30개가 안되는 방향에 버튼들을 로딩하였습니다.  

###### 렌더링, 실제 structure 분리
재생 버튼들이 부드럽게 움직이게 하고 동적으로 버튼이 추가될 때 순간 UI가 튀는 문제를 보완하기 위해 재생버튼의 위치박스들을 담은 부모와 실제 버튼들을 담은 부모로 분리하여 관리하였습니다. 실제 버튼들은 자신의 위치박스를 lerp 하여 추적하도록 하여 부드럽게 움직이도록 하였습니다.

###### 오브젝트 풀링
빈번한 버튼 생성 및 파괴에 따라 발생하는 컴퓨팅 리소스 부하를 줄이기 위해 재생 버튼과 이 버튼을 연겷하는 선들에 대해 오브젝트 풀링을 적용하였습니다.
